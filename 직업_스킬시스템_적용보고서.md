## 2. 세부직업 구조 (엑셀 시트18~21)

각 세부직업은 역할·자원·전투 아키타입·6대 지표(단일/광역/생존/기동/유틸/제어)와 Lv1~100 스킬 11~12종을 가진다.

### 2-1. 전사 (STR · 분노) — 근접 아키타입
| 세부직업 | 역할 | 정체성 |
|---|---|---|
| 검과 방패 | 탱커/수호 | 피해감소·도발·기절 (생존 5) |
| 양손검 | 광역 근접딜러 | 회전·검기·처형 (광역 5) |
| 도끼 | 버서커/폭딜 | 출혈 DoT·광폭화·흡혈 (단일 5) |
| 단도 | 듀얼리스트/치명타 | 고속·치명·은신 (기동 5) |
| 창(폴암) | 리치/견제/제어 | 관통·끌어당김·균형 (제어 4) |

### 2-2. 궁수 (DEX · 집중) — 원거리 아키타입
| 세부직업 | 역할 | 정체성 |
|---|---|---|
| 활(장궁) | 저격/단일 고뎀 | 관통·방어무시·약점 (단일 5) |
| 쇠뇌(석궁) | 관통/연사 | 일렬 관통·폭발 볼트 (광역 4) |
| 마법궁수 | 원소화살/부가효과 | 화염·냉기·전격·디버프 (제어 4) |
| 속사궁(단궁) | 초고속 연사/기동 | 다단·회피·이동사격 (기동 5) |
| 함정/조련 | 설치/소환 | 함정·소환수·광역제어 (유틸 4) |

### 2-3. 마법사 (INT · 마나) — 시전 아키타입
| 세부직업 | 역할 | 시전 스타일 |
|---|---|---|
| 불 | 고뎀/스플래시 | 메테오(낙하 광역) |
| 얼음 | 이동제한/제어 | 볼트+빙결 장판 (제어 5) |
| 번개 | 연쇄/즉발 | 체인(연쇄 번개) |
| 물 | 지속/관통/회복 | 빔(관통 물줄기)+회복 |
| 바람 | 기동/넉백 | 콘(부채꼴 다발)+넉백 (기동 5) |
| 땅 | 방어/기절/단일 | 메테오+기절 (생존 4) |
| 버프(인챈트) | 아군강화/보호막 | 볼트+자가 보호막 (유틸 5) |
| 디버프(저주) | 적약화/저주/DoT | 볼트+DoT (제어 5) |

> 학습한 스킬이 액티브 실행기에 누적된다. 예) 궁수가 「화염 화살」 학습 → 시그니처 사격에 화염·화상 부여, 「다중 사격」 → 발사 수 +1, 「관통 저격」 → 관통 부여. 패시브/궁극기도 위력에 가산.

### 밸런스 진단
1. **클래스 정체성 일치** — 1타 화력 마법사>전사>궁수, 생존 전사>마법사≈궁수(전사 VIT 우위), 궁수는 연사·다중으로 보정. 엑셀 점수 매트릭스와 부합.
2. **성장 페이싱 양호** — Lv10/20/30/70에 패시브 슬롯, Lv1~100에 스킬 11~12종이 고르게 해금. 30Lv 전후 "특색 진입"(원거리/광역 확장)도 데이터에 반영.
3. **지속 공성 텐션** — 몹 HP는 1.06^일차로 지수 증가해, 공성 일차가 매우 높아지면(수십 일+) 단일 화력을 광역·수비대·콤보·끓는 기름으로 보완해야 버티는 구조. 의도된 "성벽은 끝내 버티는 것" 설계와 일치.
4. **후속 튜닝 포인트** — 궁수 단일 1타가 상대적으로 낮아, 장궁 계열에 연사 보정 또는 약점 사격 방어무시를 실효화하면 단일 보스전 체감이 개선됨(다음 패스).

## 4. 조작 / 동선

| 키 | 동작 |
|---|---|
| 1~6 | 무기 / Q 스킬 / E 차지 스킬 / R·T 예비 |
| F | 성벽↔안마당(계단) · 성벽↔성 밖(밧줄) |
| G | 성 내부 시설(근접 시): 보급·고용·직업 사범 |
| C / I / K | 캐릭터 / 인벤토리 / 스킬창 |

**동선**: 성벽(전투) → F로 안마당 하강 → 안마당의 보급 상인·용병 고용관·직업 사범(전직+스킬 학습) 이용 → F로 성벽 복귀.
